# Render-Page

### Instalação

Clone o repositório

> https://github.com/joelton/Render-Page.git

Instale as dependências

> composer install

### Executando os testes do PHPUnit

> vendor/bin/phpunit --bootstrap src/Page.php tests/PageTest.php
